Go Libraries

This is a list of Go libraries that I have used and found helpful.

  • github.com/Shopify/sarama - Kafka client.
  • github.com/jinzhu/copier - Tools that helps copy deepcopy fields struct to struct.
  • github.com/kelseyhightower/envconfig - Tool that reads environment variables to py them to struct.
  • github.com/sirupsen/logrus - Logger.
  • github.com/stretchr/testify - Tool that helps write tests.
  • github.com/grpc/grpc-go - gRPC library to client and server.
  • github.com/jordan-wright/email - Email client library.
  • github.com/getsentry/sentry-go - Sentry SDK for go. Helps to track errors.
  • github.com/go-chi/chi - HTTP router.
  • github.com/go-playground/mold - Tool that helps to modify struct fields.
  • github.com/go-playground/validator - Tool that helps to validate struct fields.
  • github.com/go-redis/redis - Redis client for go.
  • github.com/golang-jwt/jwt - JWT generator and validator.
  • github.com/golang-migrate/migrate - Migration tool for databases.
  • github.com/google/uuid - UUID generator.
  • github.com/minio/minio-go - Amazon S3 compatible client.
  • github.com/uptrace/bun - SQL database ORM.
  • https://github.com/golang/mock - Interface mocking tools (I prefer this one).
  • https://github.com/vektra/mockery - Another mocking tool.
  • https://github.com/casbin/casbin - Authorization tool, helps to set permissions to users and roles.
